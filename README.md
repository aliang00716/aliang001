修改内容
1. 方向修正 — 东南 = 45° 斜线
使用角度函数精确计算每个方向的位移：

原文方向	角度	效果
东	0°	X轴正方向（水平右）
东南	45°	往X轴下方偏转45°（斜向右下）
南	90°	Y轴正方向（垂直下）
西南~东北	135°~315°	对应斜角
关键变化：南次二经中 "柜山→长右之山：东南四百五十里" 现在表现为一段 45° 斜向下的线段（dx=318.2, dy=318.2），总计长度恰好 450px。之前这条线段被错误地画成了水平方向。

2. 比例严格统一 — 1里 = 1像素
所有线段长度严格按照 距离 × 1px/里 计算：

「东五百里」= 500px 水平线段
「东南四百五十里」= 450px 斜线段（45°）
「东三百里」= 300px 水平线段
之前的问题：不同方向角的缩放不一致，导致有些线段看起来比实际长/短。

3. 原文准确 — 从网站抓取
所有原文数据改为从 识典古籍（shidianguji.com）抓取的《山海经》标准文本，替换了之前基于模型记忆的内容。涉及南次一经10山、南次二经17山、南次三经13山，合计40座山的完整原文。
